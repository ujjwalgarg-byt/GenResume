# 🚀 GenResume Backend — AI Resume Analyzer API

The **GenResume Backend** is a RESTful API built using Node.js and Express that powers the AI Resume Analyzer and Generator. It handles resume uploads, AI-based analysis, resume generation, and database management.

This backend processes resumes, interacts with Generative AI APIs, and stores user and resume data securely in MongoDB.

---

## 🌟 Features

- 📄 Resume Upload API (PDF/DOCX)
- 🤖 AI Resume Analysis API
- 📊 Resume Scoring System
- 🧠 Resume Improvement Suggestions
- ✨ AI Resume Generation
- 🗄️ MongoDB Database Integration
- 🔐 Environment Variable Security
- ⚡ Fast and Scalable REST API

---

## 🛠️ Tech Stack

- **Runtime:** Node.js  
- **Framework:** Express.js  
- **Database:** MongoDB  
- **ODM:** Mongoose  
- **AI Integration:** OpenAI / Gemini API  
- **File Upload:** Multer  
- **Environment Config:** dotenv  
- **Development Tool:** Nodemon
